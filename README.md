# paszmq &mdash; Delphi Pascal Bindings for ØMQ

This project provides Delphi Pascal bindings for the 2.2 release of ØMQ (ZeroMQ). 
It follows closely the APIs provided for the C and C++ bindings.

Bundled libzmq version: **2.2.1** (32 bit, compiled with VS 2010, released).
